Add social sharing links to your website.

== Description ==

Share Monkey is an easy to use plugin which allows you to add share buttons to selected post types/areas on your website.
The following features are available

* Choose default icon style or select custom color for icon background and text
* Choose any number of social networks from the available list, by drag and drop
* Drag and change the order in which they are displayed to the users
* Live preview of the icons from the settings page as you change options
* Available social networks are: Facebook,Twitter,Google,Linkedin,Pinterest,Whatsapp

== Shortcodes ==

You can include the Share Monkey bar inside post content or widgets by using the short code `[share_monkey /]`

== Installation ==

1. Upload `share-monkey` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go the the settings page for customization by clicking the Share Monkey Settings link in the left admin panel
